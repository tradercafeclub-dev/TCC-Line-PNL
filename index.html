การตั้งค่า API

บันทึกแล้ว
วิธีสร้าง Google Sheets API Key (ฟรี) 1. ไปที่ console.cloud.google.com
2. สร้างโปรเจกต์ใหม่ หรือเลือกโปรเจกต์เดิม
3. ไปที่ APIs & Services → Enable APIs → ค้นหา "Google Sheets API" → Enable
4. ไปที่ Credentials → Create Credentials → API Key
5. คัดลอก API Key มาใส่ช่องด้านบน
6. ตั้งค่า Spreadsheet ให้เป็น Anyone with the link can view
